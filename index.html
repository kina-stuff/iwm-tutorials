Object Linking

Object linking, as the name implies, links an object to another object, making the child object follow the parent.

When you use the link tool the object that you have currently selected will be a child of whatever object you click on with the link tool (which would then be the parent).

The effects of linking objects

  • The child will follow the movement of the parent
  • The child will get destroyed when the parent gets destroyed
  • The child will toggle collision when the parent toggles collision

Notes

  • A parent can have multiple children but a child cannot have multiple parents
  • Children's own movement through events will be combined with the movement of the parent, that is, if the parent is moving right and the child has an event for moving down, then the child will move diagonally down-right
